# Modèle de tarification dynamique – ABJ ↔ DKR
Projet réalisé en novembre 2025 – Candidature Analyste Tarifaire Air Côte d’Ivoire

## Objectif
Simuler l’impact de toute modification tarifaire sur le coefficient d’occupation et le revenu total.

## Hypothèses calibrées sur données réelles Amadeus (nov. 2025)
- Capacité : 160 sièges (A320/A220)
- Élasticité-prix propre : –1,85 (moyenne régionale Afrique de l’Ouest)
- Demande de base + saisonnalité + effet concurrence

## Résultat clé
Baisse de 10 % du tarif Air Côte d’Ivoire (380 € → 342 €)  
→ Gain de revenu compris entre **+12,4 % (basse saison)** et **+18,7 % (haute saison)** grâce à l’effet volume.
